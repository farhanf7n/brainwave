# Brainwave

#### Modern AI website with sleek parallax effects and bento box layouts. Made using React.js and Tailwind CSS.

![](./src/assets/Thumbnail.png)

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:farhanf7n/brainwave.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the project
   ```sh
   npm run dev
   ```
